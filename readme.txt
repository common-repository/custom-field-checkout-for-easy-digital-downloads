**Custom Field Checkout For Easy Digital Downloads** you can be add in custom field in admin side as well frontend side in EDD Chekout on show field. very simple way you need to add field just need to add field name which you want to add in [Easy Digital Downloads](https://easydigitaldownloads.com)

Ex. if you need to add phone, addition  field you can add in Custom field **Easy Digital Downloads checkout field editor**.

**Easy Digital Downloads field editor Checkout** can be add on field to action field name and placeholder

#### How To use Custom Field Checkout For Easy Digital Downloads
Go TO admin in **Custom Field For EDD Checkout Editor > Add New Field** there just you need to add field name and text field will be show automatically in Checkout page field in fronted and backend both side.
